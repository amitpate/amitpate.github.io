# amitpate.github.io

Personal website of **Amit Pate** — AI founder & serial entrepreneur building generative AI for the creative, media and entertainment industries.

🌐 **Live:** https://amitpate.github.io

Founder of **Orio** (AI 3D game generation), **PentoPix** (text-to-3D filmmaking — SXSW Pitch Best in Show 2023) and **Snaptivity** (AI-powered sports fan engagement).

## Stack
Static, dependency-free HTML + CSS — built for speed, SEO and answer-engine optimisation (JSON-LD `Person` + `FAQPage` structured data, Open Graph, Twitter Cards, sitemap, AI-crawler-friendly robots.txt).

## Files
- `index.html` — the site
- `404.html` — custom not-found page
- `sitemap.xml`, `robots.txt` — SEO/AEO
